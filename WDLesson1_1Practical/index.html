Web Technologies

HTML

HTML stands for Hypertext Markup Language. It's used to create web pages and web sites. It's commonly misunderstood to be a programming language, but it's a markup language.

CSS

CSS stands for Cascading Style Sheets. It's used with HTML to style a web page and a web site. It's not a programming language or a markup language; it's a stylesheet language.

JavaScript

JavaScript is a core technology of the web. It's used to create dynamic and interactive websites and web applications. It was created in May 1995 by Brendan Eich after he realized that web pages are too static.

Bootstrap

Bootstrap is an open-source front-end framework used for developing websites for mobile devices. It has a collection of pre-written HTML, CSS, and JavaScript components to help developers create web pages quickly and efficiently. Bootstrap is relatively easy to learn and use for people with a basic understanding of HTML and CSS.
